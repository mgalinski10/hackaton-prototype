# Animal Helper – Prototype

Aplikacja webowa wspierająca adopcję zwierząt ze schronisk. Umożliwia wyszukiwanie schronisk na interaktywnej mapie Polski, przeglądanie ich profili, ocenianie wiarygodności (AI Trust Score) oraz zgłaszanie się wolontariuszy do weryfikacji placówek. System posiada panel administracyjny do moderowania ocen i zarządzania wolontariuszami.

## Uruchomienie

### Wymagania

- Node.js >= 18
- npm

### Kroki

```bash
cd animal-helper-prototype
npm install
npm run dev
```

Aplikacja będzie dostępna pod adresem: http://localhost:3000

### Inne polecenia

```bash
npm run build   # produkcyjny build
npm run start   # uruchomienie po buildzie
```
